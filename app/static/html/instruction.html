Project Instructions


Browser Setup

  • Browser: It's the best to run the Google Chrome for this project.
  • Precise Memory Info: To measure the precise memory usage for each algorithm, please run the Google Chrome by this:
    macOS:
    1. Close all running instances of Chrome.
    2. Open Terminal inside your Application folder.
    3. In the Terminal, enter the following command: Google\ Chrome.app/Contents/MacOS/Google\ Chrome --enable-precise-memory-info
    Windows:
    1. Close all running instances of Chrome.
    2. Press Win + R to open the Run dialog. Type cmd and press Enter to open the Command Prompt.
    3. Enter the following command: start chrome --enable-precise-memory-info
    Linux:
    1. Close all running instances of Chrome.
    2. Open a Terminal window.
    3. Enter the following command: google-chrome --enable-precise-memory-info

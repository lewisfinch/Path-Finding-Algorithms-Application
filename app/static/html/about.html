Project Description

Project Title:

Finding the Shortest Paths in the University of Minnesota Twin Cities Campus Map

Project Description:

This project aims to find the shortest path between two points within the University of Minnesota Twin Cities campus map.

Tools Used

  • OpenStreetMap (OSM)
    • Data collection
  • NetworkX
    • Data visualization & manipulation
  • Python
    • Algorithms implementation

Algorithms Used

  • [AS], A* Algorithm
  • [BS], Bidirectional Search
  • [BFS], Breadth-First Search
  • [DS], Dijkstra's Algorithm

Solution Evaluation

The solutions will be evaluated based on the following aspects:

  • Length of the shortest paths
  • CPU run time, time complexity, and memory usage
  • Accuracy of the solution
  • A visual comparison of all solutions

Experiment Results

Test Mode: there is a loop for 100 times of shortest-path searching. Results with positive memory usages are considered as 'Valid' and are counting into the averages.
Each test is tested by running the test mode 20 times and taking the averages of stats. After each test on each algorithm, the browser is refreshed for the next test.


GOLD: best performance.
MAROON: worst performance.
